Binary Pulsar Constraint Lab — Method Notes

This page documents the simple sanity-check used in the Binary Pulsar Lab. It is not a full analysis, just a quick way to see whether a given QDS parameter choice \((\alpha, \lambda_c)\) would already be excluded by high-precision timing of relativistic binaries.

1. Model

We treat the QDS correction as an effective Yukawa-like modification to the Newtonian acceleration:

a_eff / a_N ≈ α · exp(-r / λ_c).

2. Constraint logic

  • Choose a binary separation \(r\) (in any convenient units).
  • Pick QDS parameters \(\alpha\) and \(\lambda_c\) in the same units.
  • Compute |α · exp(-r / λc)|.
  • Compare to an allowed fractional deviation band |Δa/a|max (typical timing limits are ≲ 0.2–0.3% for well-measured systems).

If |α · exp(-r / λc)| > |Δa/a|max, then this particular parameter point is labelled “ruled out” at that scale.

3. Status

The lab is intentionally conservative and is meant to stop “too large” QDS gravitational tweaks from being used elsewhere. A full publication would replace this toy filter with system-by-system timing fits.
