Servers & Ports

8000 / 8011 are your main local ports.

s8 server 8000

Kill old servers and start 8000 in www.
cd ~/OMEGA_EMPIRE/UV_QDS/www && kps; python -m http.server 8000

s1 server 8011

Kill old servers and start 8011 in www.
cd ~/OMEGA_EMPIRE/UV_QDS/www && kps; python -m http.server 8011

qds8000 quick 8000

Restart Python HTTP server on 8000 (bound to localhost).
pkill -f "http.server 8000" 2>/dev/null; python -m http.server 8000 --bind 127.0.0.1 &

qds8011 quick 8011

Restart Python HTTP server on 8011 (localhost).
pkill -f "http.server 8011" 2>/dev/null; python -m http.server 8011 --bind 127.0.0.1 &

battserve / battserve1 battery lab

Start servers in www for battery dashboards.
cd ~/OMEGA_EMPIRE/UV_QDS/www && python -m http.server 8000
cd ~/OMEGA_EMPIRE/UV_QDS/www && python -m http.server 8001

labside / darkside meta launchers

High-level launch scripts for lab/front-door.
/data/data/com.termux/files/home/labside.sh
/data/data/com.termux/files/home/darkside.sh

Kill / Reset

kps kill python servers

Kill all http.server processes.
pkill -f http.server 2>/dev/null || true

battkill kill 8000 / 8001

Kill battery servers on 8000 & 8001.
pkill -f "http.server 8000" 2>/dev/null;
pkill -f "http.server 8001" 2>/dev/null;
echo "[batt] server stopped"

Kill 8000 (manual) from doc

Bare kill for port 8000 (no echo). Pulled from your KILL SERVER doc.
pkill -f "http.server 8000" 2>/dev/null

batt2d battery HTML last 2 days

List all *battery*.html files modified in last 2 days.
cd ~/OMEGA_EMPIRE/UV_QDS/www && find . -type f -name "*battery*.html" -mtime -2 -print | sort

html2d / html2d_pretty HTML last 2 days

List recent HTML pages, plain or pretty (with sizes & dates).

cd ~/OMEGA_EMPIRE/UV_QDS/www && find . -type f -name "*.html" -mtime -2 -exec ls -lh --time-style=long-iso {} \; | sort -k6,7
